# Tachyon Manifold Solver

This project solves two puzzles (“Puzzle 1” and “Puzzle 2”) defined in the provided files:

- puzzle1.txt — classical tachyon manifold
- puzzle2.txt — quantum tachyon manifold
- input.txt — manifold diagram to process

## Language chosen

**Python 3.11+**

Reasons:

- Ideal for string and grid manipulation
- Clean BFS implementation with standard library
- Fast enough for the grid size
- Zero external dependencies
- Simple execution model

## Prerequisites

- Python 3.11 or later
- No additional libraries required
