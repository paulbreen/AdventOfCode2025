# Printing Department Puzzle Solver

This project solves two related grid-based puzzles involving rolls of paper (`@`) and forklift accessibility.

## Language

**Python 3** was chosen because:

- Well-suited for grid iteration and neighbor scanning
- Minimal boilerplate for file I/O
- Fast enough for the puzzle size
- No external dependencies needed

## Prerequisites

- Python 3.8+
- Standard library only

## Files

- `input.txt` — puzzle input grid  
- `solution.py` — solver  
- `puzzle1.txt` / `puzzle2.txt` — specifications (not required at runtime)
