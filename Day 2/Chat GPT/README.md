# Gift Shop – Invalid Product ID Solver

This repository contains a solution for the “Gift Shop” puzzles (Day 2), where
you are given ranges of product IDs and must detect *invalid* IDs according to
digit-repetition rules. 

## Problem Summary

### Input

- `input.txt`: a single logical line (possibly wrapped) containing comma-
  separated ranges of the form:

  ```text
  start1-end1,start2-end2,start3-end3,...
